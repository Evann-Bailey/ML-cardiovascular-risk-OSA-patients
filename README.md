# Machine Learning Prediction of Cardiovascular Risk in OSA Patients

This project uses machine learning to analyze and predict cardiovascular risk factors in patients diagnosed with Obstructive Sleep Apnea (OSA).

## Project Structure
* **`raw data/`**: Contains the original datasets used for the analysis (e.g., `capstone_raw_data.csv`).
* **`Notebooks/`**: Jupyter notebooks covering data cleaning, Exploratory Data Analysis (EDA), and model training.

##  Dataset Information
The data includes clinical variables related to OSA severity and cardiovascular health markers.
* **Primary Author:** Evann Bailey
* **Status:** Manuscript in revision / Capstone Project

##  Requirements
To run the analysis, you will need:
* Python 3.x
* Pandas
* Scikit-learn
* Matplotlib/Seaborn (for GIS and data visualization)
